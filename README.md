# DSA Pattern Mastery

This repository organizes Data Structures & Algorithms problems by pattern rather than platform.

The objective is not just solving problems, but understanding:

- Why the pattern works
- When to apply it
- How to optimize brute-force approaches
- Time & Space tradeoffs

---

## 📚 Patterns Covered

- [Sliding Window](./sliding-window)
- [Two Pointers](./two-pointers)
- Binary Search
- Graph Traversals (BFS / DFS)
- Heap / Priority Queue
- Dynamic Programming
- Intervals
- Prefix Sum

---

## 🧠 Methodology

Each problem includes:

- Problem link
- Pattern used
- Brute force intuition
- Optimized solution
- Time complexity
- Space complexity
