# Two Pointers Pattern

## 📌 When to Use

Use this pattern when:
- The input array is sorted
- You need to search for pairs
- You compare elements from both ends
- Checking symmetry (e.g., palindrome)

---

## 🧠 Core Idea

Maintain two indices:
- One starting from the left
- One starting from the right

Move pointers based on problem conditions.

---

## 📚 Problems

1. Valid Palindrome  
   - Symmetry check  
   - Time: O(n)  
   - Space: O(1)

2. Two Sum II – Input Array Is Sorted  
   - Pair search in sorted array  
   - Time: O(n)  
   - Space: O(1)
